Steps to set Wi-fi SSID/Password to match the site’s Wi-fi for provisioning and testing:

- Go to [http://192.168.1.254/](http://192.168.1.254/) while connected to the AT&T router (usually set to “heylo-provisioning”)
- Go to the “Home Network” tab. And then within the Home Network section choose the “Wi-fi” Tab.
- It will prompt for the Admin password (Device Access Code). Currently it is still set to the default `830??@97/9` but may change so confirm what the current password is before entering.
- In the “Home SSID” section set the Network Name (SSID) to the site’s SSID.
- Set the Password to the site’s password.
- Scroll down and click the `Save...` button (underneath the Guest SSID section on the page)

The Wi-fi SSID/Password will now update. Your devices connected to the old network SSID will no longer connect and their Wi-fi settings will need to be updated to connect to the new SSID with the new password.
